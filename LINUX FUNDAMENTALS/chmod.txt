Description:
chmod (short for change mode) is used in Linux/Unix to set or modify permissions for files and directories.
It controls who can read, write, or execute a file or folder.

three permission types:

Permission	Symbol	Numeric	Meaning
Read	r	4	    view file
Write	w	2	    modify file
Execute	x	1	    run/enter


Useful Flags in chmod:
1. -R —> Recursive
chmod -R 755 folder/

Applies permissions to the folder AND everything inside it
(files + subfolders)

2. -v —> Verbose
chmod -v 644 file.txt

Shows a message for each file changed, useful for tracking

3. -c —> Changes only
chmod -c 644 file.txt

Shows output only when a change actually happens (less noisy than -v)

4. -f —> Silent / Quiet
chmod -f 777 file.txt

Suppresses error messages (useful in scripts).

5. --reference=FILE
chmod --reference=original.txt newfile.txt

Copies permissions from one file to another — very convenient.

6. --help
chmod --help

Displays help text for the command.
